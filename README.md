# Ну, погоди! — Электроника ИМ-02

PWA-приложение, имитирующее культовую советскую портативную игру «Электроника ИМ-02» (1984).

## Запуск

```bash
# Через Python
python3 -m http.server 8080

# Или через npx
npx serve .
```

Откройте http://localhost:8080 в браузере.

## Установка как PWA

1. Откройте приложение в Chrome/Safari на телефоне
2. В меню выберите «Добавить на главный экран» / «Установить приложение»
3. Игра будет доступна как отдельное приложение

## Управление

- **4 кнопки** слева и справа — перемещение Волка по углам (верх-лево, низ-лево, верх-право, низ-право)
- **ИГРА А** — яйца с 3 сторон (легче)
- **ИГРА Б** — яйца с 4 сторон (сложнее)

## Правила

- Волк ловит яйца корзиной
- За пойманное яйцо: +10 очков
- За разбитое яйцо: +1 штраф
- 3 штрафа = конец игры
- При 200 и 500 очках штрафы аннулируются
